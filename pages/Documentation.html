Documentation SQL Trainer

Apprenez à utiliser le simulateur de base de données

Qu'est-ce que SQL Trainer ?

SQL Trainer est une application web qui simule une interface de gestion de bases de données comme PHPMyAdmin, mais avec :

  • Aucune installation requise - Fonctionne directement dans le navigateur
  • Stockage local - Toutes vos données sont sauvegardées dans votre navigateur
  • Interface simplifiée - Parfaite pour apprendre les bases du SQL
  • Requêtes supportées : SELECT, INSERT, UPDATE, DELETE, CREATE TABLE, etc.

Premiers pas

Connexion

Utilisez les identifiants par défaut :

Créer votre première base

  1. Cliquez sur "Nouvelle" dans la sidebar
  2. Entrez un nom (ex: "ma_premiere_bdd")
  3. Validez avec le bouton "Créer"

Gestion des tables

Créer une table

  1. Sélectionnez une base de données
  2. Allez dans l'onglet "Structure"
  3. Cliquez sur "Nouvelle table"
  4. Renseignez le nom et les colonnes
Exemple de structure :
id - INT (clé primaire)
nom - VARCHAR(255)
email - VARCHAR(255)
date_inscription - DATE

Insérer des données

Dans l'onglet SQL, exécutez :

INSERT INTO ma_table (id, nom, email) VALUES
(1, 'Pierre', 'pierre@exemple.com'),
(2, 'Sophie', 'sophie@exemple.com');

Requêtes SQL supportées

SELECT - Lire des données

-- Sélection simple
SELECT * FROM utilisateurs;

-- Avec condition
SELECT nom, email FROM utilisateurs WHERE id = 1;

-- Trier les résultats
SELECT * FROM produits ORDER BY prix DESC;

INSERT/UPDATE/DELETE - Modifier des données

-- Insertion
INSERT INTO clients (nom, ville) VALUES ('Alice', 'Paris');

-- Mise à jour
UPDATE clients SET ville = 'Lyon' WHERE nom = 'Alice';

-- Suppression
DELETE FROM clients WHERE id = 5;

Gestion des tables

-- Créer une table
CREATE TABLE produits (
id INT,
nom VARCHAR(255),
prix DECIMAL(10,2)
);

-- Supprimer une table
DROP TABLE produits;

-- Créer une base
CREATE DATABASE ma_nouvelle_bdd;

-- Supprimer une base
DROP DATABASE ancienne_bdd;

FAQ

Où sont stockées mes données ?

Toutes les données sont sauvegardées dans le localStorage de votre navigateur. Elles restent disponibles même après fermeture du navigateur, mais sont spécifiques à chaque appareil/navigateur.

Comment réinitialiser l'application ?

Exécutez cette commande dans la console de développement (F12) :

localStorage.clear();
location.reload();

Puis-je exporter mes données ?

Pas encore, mais cette fonctionnalité est prévue dans une future version !
